
Usage:
	Go to the terminal and run the command as follows.
	# python main.py filename.ext
	
	generated_cfg folder will be created and cfgs(*.dot files) will be there.
	To convert *.dot files to image, run follow command.
	#dot -Tpng xxxx.dot -o outfile.png

Requirement: 
	python version >= 3.6(developed with python 3.10 and tested in Windows)
	pyevmasm should be installed.(pip install pyevmasm)
